Hi, I'm [Deden Fathurahman](https://dedenf.com) — an [engineer](https://github.com/dedenf), tech enthusiast, [blogger](https://notes.dedenf.com), and [newsletter author](https://jakartadev.substack.com). I currently work in finance industry, building systems that handle money, data, and scale. I listen to a lot of [podcasts](/2018/01/podcast), ramble about development at [Jakartadev](https://jakartadev.com), and write about engineering, agile practices, high-performance systems, product management, and [finance](https://seputarfinansial.com).

I love coffee — I even started [Epicoffee](http://epicoffee.com) (semi-active, like most side projects). You can find my other blogs at [dedenf.com](https://dedenf.com).

### About this blog

This site started around 2013, but I've been blogging since 2006. My early content was scattered across Blogger, Blogsome, Posterous, and WordPress before I finally bought my own domain and imported everything worth saving.

The site is built with [Jekyll](https://jekyllrb.com/) and deployed on [Netlify](https://www.netlify.com/). Source code lives on [GitHub](https://github.com/dedenf/dedenf.github.io/), DNS is handled by [Cloudflare](https://www.cloudflare.com/). No server to manage, just push and go.
